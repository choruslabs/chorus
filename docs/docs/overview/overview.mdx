# Overview

Chorus is a civic crowd sourcing tool that allows members of a community to share their ideas with decision-makers.
The decision-makers run discussions on specific topics to organize opinions.
The tool analyzes the collected data and surfaces consensus ideas to aid the decision-making process.

## Definitions

- **Conversation**: The combination of the topic and the ideas submitted by community members
- **Facilitators**: the individuals who chose the topic and run the conversation (a.k.a decision-makers)
- **Participants**: the individuals who submit their ideas to the conversation (a.k.a community members)
- **Statement**: an idea that a participant contributes to the conversation
