# Statements and votes

Participants express their ideas by adding 140-character statements to the conversation.

- Each statement is a single, well-defined opinion on the topic
- **Example statement**: “We should plant more trees in the neighborhood”

Each statement is placed in a stack which all other participants can see.
Participants vote on statements in the stack - indicating whether they ‘agree’, ‘disagree’, or ‘pass’ (uncertain/neutral)
